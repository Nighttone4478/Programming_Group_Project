# 栽要報告

## 專案簡介
本專案以 C 語言搭配 Raylib 實作暗棋遊戲，內容包含棋盤顯示、棋子隨機配置、翻棋、移動、吃子、電腦對手（AI）與完整遊戲流程控制。  
遊戲依照標準暗棋規則進行，並提供玩家先手 / 電腦先手選擇、狀態提示與操作介面。

## 功能說明
- 顯示暗棋棋盤與棋子
- 棋子開局隨機排列，初始皆為背面朝上
- 可選擇玩家先手或電腦先手
- 支援翻棋、移動、吃子
- 依標準暗棋規則進行判定
- 電腦可進行基本決策判斷
- 遊戲可持續進行直到結束

## 執行方式
請先確認已安裝 Raylib 與對應編譯環境，再編譯並執行專案原始碼。

## 專案檔案
- `main.c`：主程式進入點
- `game.c / game.h`：遊戲流程控制

- `ai.c / ai.h`：電腦玩家邏輯
- `render.c / render.h`：畫面繪製
- `assets.c / assets.h`：字型與資源載入
